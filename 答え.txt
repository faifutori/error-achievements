====================================================
エラー実績解除　全10バッジの答え（ネタバレ注意）
====================================================
どの端末でも出せるものが多いですが、一番簡単な手順で書いています。
書き換えたら Ctrl+Enter で実行。失敗しても「修復」ボタンで戻せます。
エラーが出たらパネルを閉じて、空から降りてくるバッジを拾ってください。

----------------------------------------------------
1. SyntaxError = 構文崩壊者【Common】
----------------------------------------------------
端末1で、
  print("name:", name)
の最後の閉じ括弧 ) を消して実行。
（別解：端末2で if hp > 50: のコロン : を消す）

----------------------------------------------------
2. IndentationError = インデントの反逆者【Common】
----------------------------------------------------
端末2で、
  print("元気です")
の行頭のスペース4つを消して実行。
（別解：どの端末でも1行目の先頭に余計なスペースを入れる）

----------------------------------------------------
3. NameError = 幽霊変数使い【Common】
----------------------------------------------------
端末1で、
  print("name:", name)
を
  print("name:", nam)
に書き換えて実行。存在しない名前なら何でもOK。

----------------------------------------------------
4. TypeError = 型の反乱軍【Common】
----------------------------------------------------
端末3で、
  print(add(2, 3))
を
  print(add(2, "3"))
に書き換えて実行。数値と文字列は足せない。
（別解：端末4で print(len(5)) を追加）

----------------------------------------------------
5. IndexError = 境界突破者【Uncommon】
----------------------------------------------------
端末4で、
  print(items[0])
を
  print(items[9])
に書き換えて実行。リストは3個しかないので9番目は範囲外。

----------------------------------------------------
6. KeyError = 鍵なき扉の探索者【Uncommon】
----------------------------------------------------
どの端末でもいいので、次の2行を追加して実行。
  d = {"a": 1}
  print(d["b"])
辞書に "b" というキーは無い。

----------------------------------------------------
7. ImportError = 輸入禁止令の違反者【Uncommon】
----------------------------------------------------
どの端末でもいいので、1行目に次を追加して実行。
  import supermodule
そんなモジュールは存在しない。
（別解：from math import teleport）
※本物のPythonでは ModuleNotFoundError と表示されますが、
  ImportErrorの一種なのでこのバッジが解除されます。

----------------------------------------------------
8. ValueError = 値の錬金術師【Rare】
----------------------------------------------------
どの端末でもいいので、次の1行を追加して実行。
  print(int("abc"))
"abc" は数値に変換できない。

----------------------------------------------------
9. AttributeError = 属性迷宮の探索者【Rare】
----------------------------------------------------
端末4で、次の1行を追加して実行。
  items.push("bow")
Pythonのリストに push は無い（正しくは append）。

----------------------------------------------------
10. ZeroDivisionError = ゼロ除算の勇者【Epic】
----------------------------------------------------
どの端末でもいいので、次の1行を追加して実行。
  print(1 / 0)
0で割ることは数学的に定義できない。

====================================================
10個すべて集めるとステージクリア演出が流れます。
バッジの確認は Space キー（または Tab）。
取得済みバッジをクリックすると「取得の記録」が見られます。
